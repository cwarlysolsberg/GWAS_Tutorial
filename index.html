Home¶

Overview¶

This tutorial provides a step-by-step guide to performing basic Genome Wide Association Study (GWAS) analyses and accompanies our GWAS Guide paper. The aim of this tutorial is to provide a simple introduction of GWAS analyses while equipping existing users with a better understanding of the processes and implementation of mainstay GWAS tools.

The tutorial is separated into five main section.

  1. Quality Control of GWAS Data
  2. Population Stratification
  3. Association Analyses
  4. Visualizing of Association results
  5. Deep dive into underpinnings of associations

Warning

Data used in this tutorial are simulated and intended for demonstration purposes only. The results from this tutorial will not reflect the true performance of different software.

Note

echo "Tab B"

You can click on the tab to change to an alternative code (eg. to a different operation system)

Requirements¶

To follow the tutorial, you will need the following programs installed:

  1. R (version 3.2.3+)
  2. PLINK 1.9
